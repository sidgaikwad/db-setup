# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-25

### Added
- Initial release
- Neon provider support with 11 global regions
- Supabase provider support
- Railway provider support
- Local Docker PostgreSQL support
- Automatic .env file configuration
- Interactive CLI with modern prompts
- TypeScript support
- Comprehensive error handling

### Features
- Region selection with emoji flags
- Authentication handling for cloud providers
- Connection string validation
- Safe .env file updates
